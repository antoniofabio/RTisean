Description of the program: delay

This program takes a scalar time series and produces a d-dimensional delay representation of it.

Usage:

delay [Options]

Everything not being a valid option will be interpreted as a potential datafile name. Given no datafile at all, means read stdin. Also - means stdin

Possible options are:

Option Description Default
-l# number of data to use whole file
-x# ignore the first # rows 0
-c# column to read or datafile,# 1
-m# dimension of the delay vectors 2
-d# delay to use 1
-o[#] output datafile without file name: 'datafile'.del
(or stdin.del if stdin was read)
If no -o is given stdout is used
-h show these options none

Description of the Output:

The time series in delay representation.
