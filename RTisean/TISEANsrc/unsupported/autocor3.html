Three point autocorrelation statistic

autocor3 [-d# -D# -l# -x# -c# -h] file(s)

-d delay one (1)
-D delay two (2x delay one)
-l number of points (whole file)
-x number of values to be skipped (0)
-c column to be read (1 or file,#)
-h show this message
Computes 
               _         _         _
        < (y - y)*(y   - y)*(y   - y) >
            n       n-d       n-D
       --------------------------------
                        _ 3
                < (y  - y)  >
                    n
