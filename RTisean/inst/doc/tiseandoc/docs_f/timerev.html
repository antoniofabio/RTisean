Time reversal asymmetry statistic

timerev [-d# -l# -x# -c# -V# -h] file(s)

-d delay (1)
-l number of points (whole file)
-x number of values to be skipped (0)
-c column to be read (1 or file,#)
-V verbosity level (0 = only fatal errors)
-h show this message
