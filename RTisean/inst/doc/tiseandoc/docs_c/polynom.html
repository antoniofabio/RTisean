Description of the program: polynom

This programs models the data making a polynomial ansatz.

Usage:

polynom [Options]

Everything not being a valid option will be interpreted as a potential datafile name. Given no datafile at all, means read stdin. Also - means stdin

Possible options are:

Option Description Default
-l# number of data to use whole file
-x# number of lines to be ignored 0
-c# column to be read 1
-m# embedding dimension 2
-d# delay 1
-p# order of the polynomial 2
-n# number of points for the fit;
The other points are used to estimate the out of sample error 		number of data
-L# determines the length of the predicted series none
-o# output file name; without -o default is used 'datafile'.pol
-V# verbosity level
  0: only panic messages
  1: add input/output messages 		1
-h show these options none

Description of the Output:

The output file contains: The coefficients of the model, the forecast errors and if the -L flag was set the predicted points.
