Description of the program: low121

This program applies a simple low pass filter in the time domain. The filter works as follows:
x'n=(xn-1+2xn+xn+1)/4

Usage:

low121 [Options]

Everything not being a valid option will be interpreted as a potential datafile name. Given no datafile at all, means read stdin. Also - means stdin

Possible options are:

Option Description Default
-l# number of data to use whole file
-x# number of lines to be ignored 0
-c# column to be read 1
-i# number of iterations 1
-V# verbosity level (0: only save the last iteration
otherwise: save all iterations) 		otherwise
-o# output file name 'datafile'.low.n, where n is the iteration
(or stdin.low.n if stdin was read)
without -o the last iteration is also written to stdout
-V# verbosity level
  0: only panic messages
  1: add input/output messages
  2: print each iteration to a separate file 		1
-h show these options none

Description of the Output:

Depending on the the -V flag one or more files are produced. Each of the files contains one column, consisting of the filtered time series.
